# Rock Paper Scissors Game

A simple Rock Paper Scissors game built using HTML, CSS, and JavaScript.

## Features
- Player vs Computer
- Score tracking
- Interactive buttons
- Responsive design

## How to Run
1. Open the `index.html` file in your browser.
2. Click Rock, Paper, or Scissors to play.
3. The computer will make its choice and scores are updated automatically.

## Tech Used
- HTML
- CSS
- JavaScript
